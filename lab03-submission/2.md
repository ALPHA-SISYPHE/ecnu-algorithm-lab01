```cpp
void heap_sort(vector<int> &arr)
{
    int size = arr.size();
    // 先堆化
    build_heap(arr);

    for (int i = size - 1; i > 0; i--)
    {
        swap(arr[0], arr[i]);
        //这里可以直接用vector的pop
        size--;
        max_heapify(arr, 0,size);
    }
}
```
注意这里面max_heapify的参数需要传递缩减后的大小，或者：
### 当然可以用 vector 的 pop_back() 来实现堆排序！这种方式的核心思路是：每次把堆顶的最大值交换到数组末尾后，直接删除末尾的已排序元素（用 pop_back()），这样 arr.size() 就天然成为堆的有效长度，无需额外传递 heap_size 参数。
